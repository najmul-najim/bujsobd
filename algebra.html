বীজগণিতের সূত্রসমূহ

(a + b)² = a² + 2ab + b²

বিস্তারিত ব্যাখ্যা: এটি একটি বর্গ সূত্র। দুটি সংখ্যা যোগ করে তাদের বর্গ মান বের করার জন্য এটি ব্যবহার করা হয়।

উদাহরণ:

  1. (3 + 4)² = 3² + 2(3)(4) + 4² = 49
  2. (5 + 2)² = 5² + 2(5)(2) + 2² = 49
  3. (6 + 1)² = 6² + 2(6)(1) + 1² = 49
